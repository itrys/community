# ITrys 商标使用政策（Trademark Policy）

> **ITrys 是一个品牌，更是一份信任。保护商标，就是保护每一位贡献者和用户的利益。**

---

## 1. 概述

### 1.1 本政策的目的

ITrys（艾雀斯）是一个开源协作组织，其名称、标识和域名构成组织的核心品牌资产。本政策旨在：

- 明确 ITrys 商标的归属与使用权
- 规范社区成员和第三方对 ITrys 商标的使用
- 保护社区信任，防止品牌滥用或混淆
- 平衡开源开放精神与品牌保护需求

### 1.2 适用范围

本政策适用于所有使用 ITrys 商标的个人、组织和商业实体，包括但不限于：

- ITrys 社区成员与贡献者
- 入驻项目及其维护者
- 商业合作伙伴与服务商
- 第三方开发者与内容创作者

## 2. 商标定义

### 2.1 ITrys 商标清单

以下为 ITrys 拥有或主张权利的商标（统称"ITrys 商标"）：

| 类型 | 商标 | 说明 |
|------|------|------|

> 📌 以上商标均由 ITrys 组织持有，具体法律主体以商标注册信息为准。

### 2.2 双域名战略

ITrys 采用 **.org + .com 双域名模式**，参照 WordPress 开源生态：

| 域名 | 定位 | 性质 | 内容 |
|------|------|------|------|
| **itrys.org** | 开源社区 | 非营利性、社区治理 | 开源项目托管、社区协作、治理文档、合规认证 |
| **itrys.com** | 商业服务 | 商业运营、企业服务 | 托管服务、企业支持、付费功能、商业合作 |

```
┌─────────────────────────────────────────────────┐
│                 ITrys 品牌                        │
├────────────────────┬────────────────────────────┤
│   itrys.org        │      itrys.com             │
│   ────────────     │      ────────────          │
│   🏛️ 开源社区      │      💼 商业服务            │
│                    │                            │
│   • 开源项目       │      • 企业托管服务          │
│   • 社区治理       │      • 付费技术支持          │
│   • 合规认证       │      • 商业授权             │
│   • 贡献协作       │      • 培训与咨询           │
│                    │                            │
│   社区驱动         │      可持续运营             │
│   开放透明         │      商业友好               │
└────────────────────┴────────────────────────────┘
```

**核心原则**：

- **itrys.org 是根基**：所有开源项目、社区治理、合规认证均在 .org 下进行
- **itrys.com 是延伸**：商业服务基于 .org 的开源项目，回馈社区
- **两者互不隶属**：.org 由社区治理，.com 由商业实体运营，但共享 ITrys 品牌
- **商业反哺开源**：.com 的收入一部分用于支持 .org 的基础设施与社区运营

## 3. 允许的使用

### 3.1 无需申请的使用

以下情形**无需事先申请**，但须遵守本政策规定：

| 使用场景 | 说明 | 要求 |
|----------|------|------|
| **描述性引用** | 在文章、演讲中提及 ITrys | 须准确、不具误导性 |
| **项目归属声明** | "本项目托管于 ITrys" 或 "ITrys Certified 项目" | 须属实 |
| **社区活动** | 线下聚会、技术分享中使用 ITrys 名称 | 不得暗示官方主办（除非获批准） |
| **开源项目链接** | 链接至 itrys.org 或项目仓库 | 不得用于钓鱼或恶意网站 |
| **学术引用** | 论文、研究报告中引用 ITrys | 须注明来源 |
| **个人社交媒体** | 个人账号简介中提及参与 ITrys | 不得暗示官方身份 |

### 3.2 需要申请的使用

以下情形须向 ITrys 提交书面申请（trademark@itrys.org）：

| 使用场景 | 说明 | 审批方 |
|----------|------|--------|
| **商业产品/服务名称** | 在产品名中使用 ITrys（如 "ITrys Manager"） | 指导委员会 |
| **第三方托管服务** | 提供基于 ITrys 项目的商业托管 | 指导委员会 |
| **培训与认证** | 提供 ITrys 相关的培训课程或认证 | 指导委员会 |
| **线下活动冠名** | "ITrys Conf" 等冠名活动 | 指导委员会 |
| **衍生商品** | 印有 ITrys Logo 的 T 恤、贴纸等 | PMC |
| **网站/应用开发** | 开发使用 ITrys 商标的网站或应用 | PMC |

### 3.3 入驻项目的商标使用

入驻 ITrys 的项目享有以下商标使用权（无需额外申请）：

| 使用场景 | 要求 |
|----------|------|
| 展示 `ITrys Certified` 或 `ITrys Incubating` 徽章 | 须通过对应阶段审核 |
| 在项目 README 中声明 "A project under ITrys" | 须处于活跃或孵化状态 |
| 链接至 itrys.org 项目页 | 须为已入驻项目 |
| 在项目文档中使用 ITrys Logo | 须遵循 Logo 使用规范 |

**限制**：
- 不得将 ITrys 商标作为项目名称的一部分（如 "ITrys-DB" 须改为 "XXX under ITrys"）
- 不得暗示 ITrys 对项目内容承担法律责任
- 退出 ITrys 后须立即停止使用 ITrys 认证徽章

## 4. 禁止的使用

### 4.1 绝对禁止

以下行为**严格禁止**，ITrys 将采取法律手段维权：

| 禁止行为 | 说明 |
|----------|------|
| ❌ **商标抢注** | 将 ITrys、艾雀斯等名称注册为商标、域名或社交媒体账号 |
| ❌ **冒充官方** | 以 ITrys 官方名义对外发布信息、签署合同或收款 |
| ❌ **域名仿冒** | 注册 itrys 类似域名（如 itrys.net、itrýs.org）用于误导 |
| ❌ **虚假认证** | 未经审核使用 `ITrys Certified` 徽章 |
| ❌ **品牌混淆** | 使用与 ITrys Logo 高度相似的标识，可能导致混淆 |
| ❌ **恶意关联** | 将 ITrys 商标用于恶意软件、欺诈或非法活动 |
| ❌ **歪曲损害** | 以损害 ITrys 声誉的方式使用商标 |

### 4.2 限制使用

以下行为需特别审慎，可能被要求停止使用：

| 限制行为 | 说明 |
|----------|------|
| ⚠️ **Fork 命名** | Fork 项目不得使用 ITrys 商标作为项目名（如 "ITrys-Fork"） |
| ⚠️ **商业推广** | 在商业推广中不得暗示 ITrys 官方推荐或背书 |
| ⚠️ **Logo 变形** | 不得对 ITrys Logo 进行变色、拉伸、旋转等修改 |
| ⚠️ **组合商标** | 不得将 ITrys 商标与自有商标组合为新商标 |

## 5. Logo 使用规范

### 5.1 标准用法

- Logo 须保持**原始比例**，不得拉伸或压缩
- Logo 周围须保留**最小留白**（Logo 高度的 1/4）
- Logo 须在**可辨识**的尺寸下使用，最小高度 24px

### 5.2 配色规范

| 场景 | 背景 | Logo 版本 |
|------|------|-----------|
| 浅色背景 | 白色/浅灰 | 标准色 Logo |
| 深色背景 | 深色/黑色 | 反白 Logo |
| 印刷品 | — | CMYK 版本 |

### 5.3 禁止的 Logo 用法

- ❌ 改变 Logo 颜色（除非使用官方反白版本）
- ❌ 在 Logo 上添加阴影、渐变或特效
- ❌ 将 Logo 置于复杂图案或低对比度背景上
- ❌ 对 Logo 进行旋转、倾斜或透视变换
- ❌ 仅使用 Logo 的一部分

> 📌 Logo 素材文件将在组织正式运营后发布于 itrys.org/brand。

## 6. .org 与 .com 品牌使用规范

### 6.1 itrys.org — 社区品牌使用

**定位**：开源社区的权威入口

**允许**：
- 开源项目仓库、文档、CI/CD
- 社区治理（GOVERNANCE.md、行为准则等）
- 合规认证流程与报告
- 社区论坛、讨论区
- 开源活动与推广

**禁止**：
- 在 .org 上提供付费服务或收费内容
- 在 .org 上投放商业广告（赞助商致谢除外）
- 在 .org 上收集用户付费信息

### 6.2 itrys.com — 商业品牌使用

**定位**：商业服务的专业入口

**允许**：
- 企业级托管服务与 SLA
- 付费技术支持与咨询
- 商业授权与许可
- 培训与认证课程
- 企业合作与赞助信息

**要求**：
- 须在 .com 首页显著位置声明与 .org 的关系
- 须明确标注付费服务的商业性质
- 不得暗示 .org 社区受 .com 商业实体控制
- 须遵守本商标政策的所有规定

### 6.3 关系声明

在 itrys.com 及相关商业材料中，须包含以下声明（或等效表述）：

> **ITrys 开源社区（itrys.org）由社区独立治理。itrys.com 提供的商业服务基于 ITrys 开源项目，但不代表或控制社区决策。**

在 itrys.org 上，须包含以下声明（或等效表述）：

> **ITrys 商业服务（itrys.com）由独立商业实体运营，部分收入用于支持社区基础设施建设。社区治理不受商业实体影响。**

## 7. 商标授权申请

### 7.1 申请流程

```
提交申请 → 初审（7 天）→ 评估（14 天）→ 批准/拒绝 → 签署协议 → 使用
```

### 7.2 申请方式

发送邮件至 **trademark@itrys.org**，包含以下信息：

```markdown
## 申请人信息
- 姓名/组织名称：
- 联系方式：
- 网站/项目地址：

## 商标使用计划
- 拟使用的商标：
- 使用场景描述：
- 使用期限：
- 是否涉及商业用途：

## 补充说明
- 任何有助于评估的信息
```

### 7.3 评估标准

| 标准 | 说明 |
|------|------|
| **不混淆** | 使用不会导致公众误认为 ITrys 官方 |
| **不损害** | 使用不会损害 ITrys 品牌声誉 |
| **不排他** | 使用不会排斥其他合规使用者 |
| **一致性** | 使用与 ITrys 价值观和社区准则一致 |

### 7.4 授权期限

| 授权类型 | 期限 | 续期 |
|----------|------|------|
| 非商业使用 | 1 年 | 到期前 30 天申请 |
| 商业使用 | 1 年 | 到期前 60 天申请 |
| 入驻项目 | 项目活跃期间 | 随项目状态自动延续 |

## 8. 侵权处理

### 8.1 举报方式

如发现 ITrys 商标被侵权使用，请联系：

- 📧 **trademark@itrys.org**
- 📧 **legal@itrys.org**

### 8.2 处理流程

```
举报 → 评估（7 天内）→ 警告函 → 协商整改 → 法律行动
```

1. **评估**：确认是否构成侵权
2. **警告函**：向侵权方发出停止侵权通知
3. **协商整改**：给予 14 天整改期
4. **法律行动**：协商无果后采取法律手段

### 8.3 善意使用

对于无意侵权的善意使用者，ITrys 将优先采取友好协商方式解决，而非直接法律行动。
